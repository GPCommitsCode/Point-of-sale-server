# Point_of_sale_server


## Steps to run server

1. Compile server code

    g++ point_of_sale_server.cpp -o server

2. Run server
    ./server <port_number>


## Steps to run client

1. Compile client code

    g++ point_of_sale_client.cpp -o client

2. Run client
    ./client <ip_address_of_server> <port_number>
